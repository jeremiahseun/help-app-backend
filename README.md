# Help App - Backend

A Next.js REST API for booking on-demand services (plumbing, cleaning, etc).

## Tech Stack

- Next.js API Routes
- TypeScript
- PostgreSQL + Prisma ORM
- JWT Authentication
- bcrypt password hashing
- Zod input validation
- Swagger (OpenAPI) docs
